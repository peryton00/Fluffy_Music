## Play Store Submission Steps
1. Go to [play.google.com/console](https://play.google.com/console)
2. Pay the one-time $25 developer registration fee (~₹2,100)
3. Create a new app → choose "App" type and "Free" pricing
4. Upload the signed `.aab` file (Android App Bundle)
5. Fill in the store listing using the content above
6. Upload screenshots and feature graphic
7. Set content rating: answer the Content Rating Questionnaire
8. Set target audience: Everyone
9. Submit for review — typically takes 1–3 business days for new apps
